API Documentation

Here are the details of the available API routes:

Method Path Description
POST /refood/{idLimbah} Add a new refood processing method
GET /refood Get all refood data
GET /refood/{idLimbah} Get refood data by idLimbah
PUT /refood/{idLimbah}/{idPengolahan} Edit a refood processing method
DELETE /refood/{idLimbah}/{idPengolahan} Delete a refood processing method
